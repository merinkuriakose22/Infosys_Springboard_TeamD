# Customer Feedback Analysis System  
### Infosys Springboard Virtual Internship – Team D

---

## 📌 Overview
This project focuses on analyzing large-scale customer feedback data to extract actionable insights using Natural Language Processing (NLP).

A dataset of **5000+ customer feedback entries** was processed through a structured pipeline including data cleaning & preprocessing, sentiment analysis, keyword extraction, and an interactive dashboard using Streamlit.

---

## 🎯 Problem Statement
Organizations receive large volumes of customer feedback, making manual analysis inefficient, time-consuming, and error-prone.

This project aims to:
- Automate feedback analysis
- Identify customer sentiment
- Detect key issues and recurring themes
- Provide a visual interface for data-driven decision-making
